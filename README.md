Repo Info
=========
Alpine [Docker] image with Java JRE and [Ansible] installed

Consuming
---------
`latest`
```
FROM mrlesmithjr/alpine-ansible-java
```
`openjdk-8-jre`
```
FROM mrlesmithjr/alpine-ansible-java:openjdk-8-jre
```
`openjdk-7-jre`
```
FROM mrlesmithjr/alpine-ansible-java:openjdk-7-jre
```
